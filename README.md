# Hobby-Megaman-X4-Switch-Between-X-and-Zero-Save-File-Editing-Mod
This is a mod that allows you to edit the save files and easily switch between X and Zero. It can easily edit save files. Made in python. I made an ez GUI package just for this that allows a graphical user interface, and a save file comparer that can work for any game that doesn't compress save files.
